# Security Policy

## Supported Versions

To help facilitate enterprise use and provide stability, the Toit community intends to support releases of Toit from the past year. 

While there is no hard guarantee, we make a best-effort commitment to maintain these older versions unless support for a specific release is explicitly revoked.

| Release Window             | Supported          |
| -------------------------- | ------------------ |
| Released within the past 12 months | :white_check_mark: |
| Released over 12 months ago        | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Toit or Jaguar, please do not open a public GitHub issue. 

Instead, please reach out to one of the [Toitlang members](https://github.com/orgs/toitlang/people) directly via our [Discord server](https://discord.gg/Q7Y9VQ5nh2). 

Please provide the details of the vulnerability privately via a direct message to a maintainer. We will try to respond as quickly as possible to assess the report and discuss the timeline for a fix.
